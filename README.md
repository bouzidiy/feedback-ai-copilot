# Feedback AI Copilot

AI-powered feedback analysis platform built with Spring Boot, Angular and PostgreSQL.

## Goals

- Learn modern Spring Boot
- Learn Angular 20
- Learn Spring AI
- Learn PostgreSQL and pgvector
- Learn AWS and Terraform
- Build a production-ready portfolio project

## Tech Stack

### Backend

- Java 21+
- Spring Boot 4
- Spring AI
- PostgreSQL
- Flyway

### Frontend

- Angular 20
- PWA
- IndexedDB

### Infrastructure

- Docker
- GitHub Actions
- Terraform
- AWS
